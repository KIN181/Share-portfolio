<h>Shareポートフォリオについて<h2/><br>
<p>    
ライフスタイルを共有してユーザーがお互いにコニュニケーションを手軽に撮ることができるアプリケーション<br />
機能としては、オーナー（管理者）、ユーザーで表示ページが異なってきます。<br />
ユーザーはライフスタイルの投稿と削除、ライクとプロフィール編集が可能です。<br />
オーナーは管理画面機能が閲覧可能です。<br />
オーナー自身はアプリ利用者を管理でき、ユーザーのアプリケーション利用の権限や投稿の内容を管理
できます。<br/>
このアプリケーショではフォロー機能で一度気になったユーザーをフォローし、再度アプリケーションに
戻ってきた際には、迅速に見つけることができます。
